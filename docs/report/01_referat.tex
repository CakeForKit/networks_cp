Цель работы~--- разработка HTTP-сервера для отдачи статического содержимого на основе архитектуры prefork с использованием системного вызова poll().

В рамках курсовой работы разработан HTTP-сервер для отдачи статического содержимого на основе архитектуры prefork с использованием системного вызова poll() и проведено нагрузочное тестирование для определения максимального количества обслуживаемых сетевых соединений и скорости отдачи данных по каждому сетевому соединению и совокупной.
