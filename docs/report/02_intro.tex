\ssr{ВВЕДЕНИЕ}

Целью данной работы является разработка HTTP-сервера для отдачи статического содержимого на основе архитектуры prefork с использованием системного вызова poll().

Для достижения поставленной цели требуется решить следующие задачи:
\begin{enumerate}[label={\arabic*)}]
	\item Проанализировать протокол HTTP/1.1 и методы GET, HEAD, а также коды состояний 200, 403, 404, 405;
	\item Спроектировать архитектуру сервера, соответствующую варианту: prefork с использованием poll() для мультиплексирования входящих соединений в каждом процессе;
	\item Разработать веб-сервер с архитектурой prefork + poll();
	\item Провести нагрузочное тестирование сервера для оценки максимального количества обслуживаемых соединений, скорости отдачи данных на соединение и совокупной пропускной способности;
\end{enumerate}

\clearpage
